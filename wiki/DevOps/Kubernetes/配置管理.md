# 配置管理：ConfigMap / Secret

## 定义

- **ConfigMap**：存储非敏感配置数据（环境变量、配置文件）
- **Secret**：存储敏感数据（密码、API Key、TLS 证书），Base64 编码存储

## 核心原理

### ConfigMap 使用方式

```yaml
apiVersion: v1
kind: ConfigMap
metadata:
  name: laravel-config
data:
  APP_ENV: "production"
  APP_DEBUG: "false"
  LOG_CHANNEL: "stderr"
  CACHE_DRIVER: "redis"
  QUEUE_CONNECTION: "redis"
  # 也可以挂载为文件
  nginx.conf: |
    server {
        listen 80;
        root /var/www/html/public;
        index index.php;
        location ~ \.php$ {
            fastcgi_pass 127.0.0.1:9000;
            fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        }
    }
```

### Secret 使用方式

```yaml
apiVersion: v1
kind: Secret
metadata:
  name: laravel-secret
type: Opaque
data:
  # Base64 编码
  DB_PASSWORD: cGFzc3dvcmQxMjM=    # password123
  REDIS_PASSWORD: cmVkaXMxMjM=      # redis123
  APP_KEY: YmFzZTY0OnNlY3JldA==    # base64:secret
```

### 注入方式

```yaml
# 方式 1：环境变量注入
env:
- name: DB_PASSWORD
  valueFrom:
    secretKeyRef:
      name: laravel-secret
      key: DB_PASSWORD
- name: APP_ENV
  valueFrom:
    configMapKeyRef:
      name: laravel-config
      key: APP_ENV

# 方式 2：挂载为文件
volumeMounts:
- name: config-volume
  mountPath: /etc/nginx/conf.d
volumes:
- name: config-volume
  configMap:
    name: laravel-config
    items:
    - key: nginx.conf
      path: default.conf
```

### ConfigMap 热更新

ConfigMap 挂载为文件后可以热更新（无需重启 Pod），但环境变量注入需要重启：

```yaml
# 推荐方式：挂载为文件 + SHA256 注解触发滚动更新
annotations:
  checksum/config: {{ include (print $.Template.BasePath "/configmap.yaml") . | sha256sum }}
```

## Laravel .env 与 K8s 配置治理

**推荐实践：**
1. 将 Laravel `.env` 中的非敏感配置放入 ConfigMap
2. 将敏感配置（DB_PASSWORD、APP_KEY、第三方 API Key）放入 Secret
3. 使用 Helm `values.yaml` 分层管理不同环境的配置差异
4. 禁止将 Secret 直接提交到 Git（使用 Sealed Secrets 或 External Secrets Operator）

## 常见问题

### ConfigMap 更新后 Pod 没生效
- 环境变量注入不会自动更新，需要重启 Pod
- 文件挂载会自动更新（可能有延迟，约 60-90 秒）
- 推荐使用 `checksum/config` 注解触发滚动更新

### Secret 被 Base64 解码看到明文
- Base64 不是加密，只是编码
- 使用 Sealed Secrets 加密后才能提交到 Git
- 或使用 External Secrets Operator 从 Vault/AWS Secrets Manager 拉取

### ConfigMap/Secret 大小限制
- 单个 ConfigMap/Secret 最大 1MB（etcd 限制）
- 大型配置文件考虑挂载为 PV 或使用 InitContainer 下载
