# Eloquent 类型转换与访问器

## 定义

- **Casts**：自动将数据库字段转换为 PHP 类型
- **Accessors**：自定义模型属性的读取逻辑
- **Mutators**：自定义模型属性的写入逻辑

## Casts

```php
class Order extends Model {
    protected $casts = [
        'total'       => 'decimal:2',
        'is_paid'     => 'boolean',
        'paid_at'     => 'datetime',
        'metadata'    => 'array',
        'status'      => OrderStatus::class,  // PHP 8.1 Enum
    ];
}
```

### PHP 8.1 Enum Cast

```php
enum OrderStatus: string {
    case Pending = 'pending';
    case Paid = 'paid';
    case Shipped = 'shipped';
}

// 自动转换
$order->status = OrderStatus::Paid;
// 数据库存 'paid'
```

### 自定义 Cast

```class EncryptedString implements CastsAttributes {
    public function get($model, $key, $value, $attributes): string {
        return decrypt($value);
    }
    public function set($model, $key, $value, $attributes): string {
        return encrypt($value);
    }
}
```

## Accessors

```php
class User extends Model {
    // Laravel 9+ 语法
    protected function fullName(): Attribute {
        return Attribute::make(
            get: fn () => "{$this->first_name} {$this->last_name}",
        );
    }

    // Laravel 8 语法
    public function getFullNameAttribute(): string {
        return "{$this->first_name} {$this->last_name}";
    }
}

// 使用
echo $user->full_name;  // 自动调用 accessor
```

## Mutators

```php
class User extends Model {
    protected function password(): Attribute {
        return Attribute::make(
            set: fn ($value) => Hash::make($value),
        );
    }
}

$user->password = 'secret';  // 自动 Hash
```

## 踩坑记录

- **Cast 性能**：`array` cast 每次读取都 json_decode，大字段用 accessor 手动控制
- **Accessor 无限循环**：accessor 内访问自身属性触发递归
- **Enum Cast 默认值**：Enum 不包含数据库值时抛异常，需处理脏数据

## 常见问题

**Q: Cast 和 Accessor 什么时候用哪个？**
A: Cast 适合类型转换（string→int, json→array）；Accessor 适合计算属性（拼接、格式化）。
