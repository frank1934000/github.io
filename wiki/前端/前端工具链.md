# 前端工具链

## 定义
前端工具链是指代码格式化（Prettier/Biome）、代码检查（ESLint/Biome）、类型检查（TypeScript）等工具的组合，用于保障代码质量和团队协作一致性。

## 核心原理

### Biome：下一代工具链
Biome 是用 Rust 编写的高性能前端工具链，用一个工具替代 ESLint + Prettier：

| 工具 | 功能 | 速度 |
|------|------|------|
| ESLint | 代码检查 | 慢（JS 实现） |
| Prettier | 代码格式化 | 慢（JS 实现） |
| **Biome** | 检查 + 格式化 | **快 10-100x**（Rust 实现） |

### Biome 配置
```json
// biome.json
{
  "formatter": {
    "enabled": true,
    "indentStyle": "space",
    "indentWidth": 2,
    "lineWidth": 100
  },
  "linter": {
    "enabled": true,
    "rules": {
      "recommended": true,
      "correctness": {
        "noUnusedVariables": "error"
      }
    }
  },
  "javascript": {
    "formatter": {
      "quoteStyle": "single",
      "semicolons": "asNeeded"
    }
  }
}
```

### TypeScript 配置
```json
// tsconfig.json
{
  "compilerOptions": {
    "target": "ESNext",
    "module": "ESNext",
    "moduleResolution": "bundler",
    "strict": true,
    "jsx": "preserve",
    "paths": {
      "@/*": ["./src/*"]
    }
  },
  "include": ["src/**/*.ts", "src/**/*.vue"]
}
```

### Mock 服务器（Mockoon）
在前后端分离开发中，Mock 服务器提供 API 模拟数据：
- Mockoon：本地 GUI 工具，可视化配置 API
- MSW（Mock Service Worker）：拦截网络请求
- json-server：快速搭建 REST API

## 常见问题

**Q: ESLint 9 扁平配置还需要 Biome 吗？**
A: ESLint 9 改进了很多，但 Biome 速度优势明显。新项目推荐 Biome，老项目可保持 ESLint。

**Q: TypeScript strict 模式太严格怎么办？**
A: 可以逐步开启，先用 `noImplicitAny`，再逐步加 `strictNullChecks` 等。
