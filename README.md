## 📖 关于这个博客

基于 [Hexo](https://hexo.io) + [Aurora 主题](https://aurora.tridiamond.tech) 搭建的个人技术博客，使用 GitHub Pages 自动部署。

> 写作不为输出，只为「把我搞懂的东西，用未来的我看得懂的方式写下来」。

## 🏃 本地运行

```bash
# 克隆仓库
git clone https://github.com/frank1934000/frank1934000.github.io.git
cd frank1934000.github.io

# 安装依赖
npm install

# 本地预览
npm run server

# 构建静态文件
npm run build
```

## 📝 写作规范

- 文章存放在 `source/_posts/<分类>/` 目录下
- 使用 Markdown 格式，支持 frontmatter 配置
- 图片统一放在 `source/images/` 目录
- 提交后 GitHub Actions 自动构建部署
